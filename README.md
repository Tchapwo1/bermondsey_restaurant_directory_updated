# Bermondsey Restaurant Directory - Organized Image Set

This reorganized version applies the structure requested:

1. Added `cover.jpg` to each restaurant folder.
2. Added `info.json` metadata to each restaurant folder.
3. Grouped images into category folders such as `food`, `interior`, `exterior`, `chef`, `people`, `branding`, `menu`, `drinks`, `press`, and `archive`.
4. Standardized image copies to a maximum long edge of 1600px while preserving aspect ratio.
5. Checked for obvious duplicate / near-duplicate images; none were clearly redundant enough to remove automatically.
6. Added numeric restaurant IDs in folder names: `001_slug`, `002_slug`, etc.
7. Added a master index file: `restaurants.json`.
8. Renamed files to the pattern: `<restaurant>-<category>-<number>.<ext>`.

Note:
- This package contains organized copies of the original files.
- Category labels are based on visual inspection.
- Restaurant factual metadata was not verified online.


Data files
----------
restaurants.json - structured restaurant metadata
restaurants.geojson - geospatial dataset
 dataset_info.json - dataset metadata
